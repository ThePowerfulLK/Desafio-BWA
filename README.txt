O código foi feito utilizando um ambiente virtual.
Para um bom funcionamento do código, verifique se o 'Pandas' e o 'Selenium' estão devidamente instalados e atualizados.
Versão do Python 3.10.2



O script faz o trabalho de varrer um arquivo '.xlsx' e transcrever os dados para um formulário web. colocando cada um
dos dados em seu devido lugar. Fazendo a automação de implementação de dados para diversas situações, como o preenchimento
de formulários de cadastro, geração de notas fiscais, e etc.
